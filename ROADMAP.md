# Roadmap

## Milestone 0 — Foundation

- Create repository.
- Commit product brief and architecture notes.
- Add initial app skeleton.
- Add Prisma data model.
- Add reader UI mock.

## Milestone 1 — PDF scholarly workflow

Definition of done:

- Document registration works.
- PDF reader loads a sample document.
- User can define page map offset.
- User can create highlight with one of ten colors.
- User can add note.
- User can generate Chicago/SBL footnote with mapped book page locator.
- User can persist and recover annotation.

## Milestone 2 — Text formats

- TXT ingestion.
- Markdown ingestion.
- DOCX ingestion path.
- Snapshot generation policy.

## Milestone 3 — Citation library

- CSL JSON source editor.
- Import/export CSL JSON.
- BibTeX/BibLaTeX export.
- Zotero-compatible data path.

## Milestone 4 — Research threads

- Add annotations to research threads.
- Export thread to Markdown and Word-compatible format.
- Preserve citations in export.

## Milestone 5 — Interrogation

- Exact search across documents, notes, tags, and citations.
- Semantic search over extracted passages.
- Grounded answer generation with cited source passages only.
